Python3的安装

Linux（Ubuntu,Debian,Deepin）下安装Python3：

安装基础库：

sudo apt-get install -y python3-dev build-essential libssl-dev libffi-dev libxml2 libxml2-dev libxslt1-dev zlib1g-dev libcurl4-openssl-dev

安装python3：

sudo apt-get install -y python3

安装pip3：

sudo apt-get install -y python3-pip

请求库的安装（Requests,Selenium,aiohttp）

抓取页面过程中，需要模拟浏览器向服务器发送请求。

Requests安装：

pip3 install requests

Selenium安装：

pip3 install selenium

Selenium是一个自动化测试工具，利用它可以驱动浏览器做特定的动作，如点击，下拉等操作。
既然Selenium可以驱动浏览器，所以我们需要浏览器来配合操作，这里推荐常用的三种浏览器。

Chrome浏览器(ChromeDriver):

1.打开Chrome，菜单“帮助”，关于Google。 Chrome，查找Chrome的版本号，我的是62.0。

2.下载ChromeDriver,找到Chrome对应的ChromeDriver版本下载,解压。
ChromeDriver版本下载
Chromedriver与Chrome的对应关系表

3.进入解压之后的ChromeDriver所在文件位置，将ChromeDriver移入/usr/bin下。
sudo mv chromedriver /usr/bin

4.将ChromeDriver配置到$PATH，修改~/.profile文件。命令如下
cd /usr/bin
vim ~/.profile
将export PATH="$PATH:/usr/local/chromedriver"写入~/.profile中。vim操作自行百度下，简单的。

5.接来下运行。
source ~/.profile

6.配置完成后，运行chromedriver，若控制台出现Only local connections are allowed.证明环境变量已经配置完成。
chromedriver

Firefox浏览器(GeckoDriver)：

1.GeckoDriver下载根据自己的系统和位数选择对应的驱动下载，例如64位的Linux 下载geckodriver-v0.20.1-linux64.tar.gz

2.进入解压之后的GeckoDriver所在文件位置，将GeckoDriver移入/usr/bin下。
sudo mv geckodriver /usr/bin

3.将GeckoDriver配置到$PATH，修改~/.profile文件。命令如下
cd /usr/bin
vim ~/.profile
将export PATH="$PATH:/usr/local/geckodriver"写入~/.profile中。

4.接来下运行。
source ~/.profile

5.配置完成后，运行geckodriver。没有报错即配置成功。
geckodriver

PhantomJS(优势：无界面，可脚本编程，Selenium也支持PhantomJS):

1.PhantomJS下载根据自己的系统和位数选择对应的驱动下载。

2.其他步骤与上面的Chrome，Firefox插件下载方法一致。

aiohttp安装：

requests库是一个阻塞式HTTP请求库，当我们发出一个请求后，它会一直等待服务器响应，得到响应后，才会进行下一步的处理。这个过程比较耗费时间。
aiohttp是一个提供异步Web服务的库，借助python3.5中的async/await关键字，异步操作更加简洁。
pip3 install aiohttp

另外，官方还推荐安装如下两个库，字符编码检测库cchardet,加速DNS解析库aiodns。
pip3 install cchardet aiodns
